Melica HTML Theme

Introduction

Melica is minimal HTML5 Blog Template, that suits great for fashion & travel bloggers. Fully responsive & retina ready, looks great on all devices. 8 HTML Files in pack (homepage, category page, standard post, full width post, 404 page, typography, about me, contact me).

Release date: 04.04.15
Author: WPHunters
Email: bbdesign_sp@yahoo.com
Used software/languages: CodeKit, LESS 1.7.5, Kit lang, jQuery 1.11.1, Twitter Bootstrap 3.3.0
Supported browsers: IE8, IE9, IE10, IE11, Firefox, Safari, Opera, Chrome
Thank you for purchasing Melica template.
If you enjoy - please rate our product ;)

Grid System

Template is divided into rows and columns, with respective classes .row and .col-*, where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12).

Full-width example

<div class="row">
    <div class="col-xs-12">
        block content
    </div>
</div>

3 equal columns example

<div class="row">
    <div class="col-xs-4">
        1st col
    </div>

    <div class="col-xs-4">
        2nd col
    </div>

    <div class="col-xs-4">
        3rd col
    </div>
</div>

2 columns with different width

<div class="row">
    <div class="col-xs-4">
        1st col
    </div>

    <div class="col-xs-8">
        2nd col
    </div>
</div>

Note More info at Twitter Bootstrap documentation page

Block Grid System

Block grids give you a way to evenly split contents of a list within the grid. If you wanted to create a row of five images or paragraphs that need to stay evenly spaced no matter the screen size, the block grid is for you.

Use a simple block-grid-*-* class to code up the block grid and specify the number of items in a row.

<ul class="block-grid-xs-3"> { x1-x6, UP TO SIX COLUMNS }
    <li>content</li>
    <li>content</li>
    <li>content</li>

    <!-- Next row -->

    <li>content</li>
    <li>content</li>
    <li>content</li>
</ul>

Use additional classes to specify a different number of items in a row for each screen size.

<ul class="block-grid-xs-1 block-grid-md-3 block-grid-lg-4">
    <!-- 1 column for mobile, 3 for tablets, 4 for desktop -->

    <li>content</li>
    <li>content</li>
    <li>content</li>

    <!-- Next row -->

    <li>content</li>
    <li>content</li>
    <li>content</li>
</ul>

    <img src="http://s2.micp.ru/922C8.jpg" alt=""/>
</div>

Setup layout

As you can see, we have attribute data-layout - here we can tune our layout:

  • data-layout="2331" - 1st row has 2 images, 2nd row has 3 images, 3rd row has 3 images, and 4th row has 1 image. Total of 9 images.
  • data-layout="13" - 1st row has 1 image and 2nd row has 3 images.

Note One row can not contain more than 4 images at once.

Mailchimp Integration

As you can see on the right sidebar - there is a simple subscription form. This form based on Mailchimp service.

How to set up

  1. Firstly, you need to register an account in mailchimp site.
  2. Then, log-in to your account, go to your dashboard and create new list(feel free to input any data in provided fields, it does not affect to subscription script).
  3. On list page click "Signup forms" link(in top menu), then select "General forms"
  4. Select "Signup form" in dropdown menu and remove all fields from this form, except "Email Address" field.
  5. Open your form in new window and view source of page(right click on page -> "View Source") and find something like this: Copy text marked with yellow marker to form on your site. Green text copy to action attribute of your form.
  6. At the end you should have something like this: 
    <form id="subscribe_form" class="form-inline" action="http://bbdesign.us9.list-manage.com/subscribe/post" method="post">
    <input type="hidden" name="u" value="1368c473b42d831e481262a9c">
    <input type="hidden" name="id" value="1b59f188b7">
    <div class="hidden-form-view"><input type="text" name="b_1368c473b42d831e481262a9c_1b59f188b7" value="" tabindex="-1"></div>

    { other form code }
</form>
  7. That's all! Check your form and enjoy! :)

Image Sliders

Theme has build-in support for photo sliders(based on slick plugin). You can easily create your own image gallery:

<section class="slick photo-slider" data-slider>
    <article><img src="images/photo1.jpg" alt=""/></article>
    <article><img src="images/photo2.jpg" alt=""/></article>
    <article><img src="images/photo3.jpg" alt=""/></article>
</section>

Note If you add vertical class to the photo-slider element, slider will tick vertical.

Note To enable navigation arrows(right, left), just add class with-arrows to main slider element as described above.
